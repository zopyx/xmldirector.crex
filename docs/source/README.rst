xmldirector.crex
================

Integration of Plone, XML Director with C-REX (http://www.c-rex.net/app/).

Requirements
------------

- Plone 4.3 (tested)
  
- Plone 5.0 (experimental, in progress)

- XML Director (xmldirector.plonecore)
